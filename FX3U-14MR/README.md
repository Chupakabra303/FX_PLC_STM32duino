MCU: STM32F103VCT6 или клон

Начальный метод прошивки: BOOT0 + USART, либо BOOT0 + SWD, либо SWD + Hardware reset.

Подробная статья: https://habr.com/ru/post/713020/

Распиновку входов-выходов см. в примерах (examples).
