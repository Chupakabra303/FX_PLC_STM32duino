# Расширенный пример на STM32duino для ПЛК JL1N-10MT (FX1N-10MT)

MCU: APM32F030C8T6 (STM32F030C8T6)

+ DI/DO/AI/AO
+ Power monitor
+ Функция перехода в загрузчик bootloaderInit()
+ Modbus RTU Slave на RS-485
